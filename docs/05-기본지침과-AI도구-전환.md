# 기본 지침과 AI 도구 전환

> 작성일: 2026-03-15
> 의사결정: Cursor Rules → AGENTS.md + Skill 베이스 전환

## 배경

피치 솔루션 프로젝트들은 초기에 Cursor의 `.cursor/rules/` 디렉토리에
프로젝트별 코딩 규칙을 관리했다.

## 전환 이유

1. **도구 종속성 제거**: `.cursor/rules/`는 Cursor 전용. Claude Code, Codex CLI 등에서 인식 불가
2. **AGENTS.md 통합**: 모든 AI 도구가 인식하는 표준 파일로 규칙 일원화
3. **스킬 베이스 작업**: 코딩 규칙 + 실행 절차가 스킬에 통합되어 rules 파일 불필요
4. **유지보수 단일화**: rules 파일과 AGENTS.md 이중 관리 → AGENTS.md 단일 관리
