# 스킬 이름 정리

`peach-skills` 저장소에서 사용하는 공개 스킬 이름 정리 문서입니다.

## 네이밍 원칙

- 공개 저장소 충돌을 피하기 위해 모든 스킬 이름에 `peach-` prefix를 사용한다.
- 저장소명은 브랜드 단위로 관리하고, 스킬명은 설치 가능한 공개 식별자로 관리한다.
- 기존 `team-*` 스킬은 통합하여 `peach-agent-team`, `peach-agent-team-refactor`로 정리한다.

## 현재 스킬 목록

### 추가 계열

- `peach-add-api`
- `peach-add-cron`
- `peach-add-print`

### 생성 계열

- `peach-gen-backend`
- `peach-gen-db`
- `peach-gen-design`
- `peach-gen-feature-docs`
- `peach-gen-prd`
- `peach-gen-store`
- `peach-gen-ui`

### 리팩토링 계열

- `peach-refactor-backend`
- `peach-refactor-frontend`

### 팀 조율 계열

- `peach-agent-team`
- `peach-agent-team-refactor`

## 설치 예시

```bash
# 단일 스킬 설치
npx skills add peachSolution/peach-skills --skill peach-gen-backend -a claude-code

# 팀 스킬 설치
npx skills add peachSolution/peach-skills --skill peach-agent-team -a claude-code -a codex -a openclaw
```
